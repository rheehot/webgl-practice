My WebGL Practice
========

![](example/screenshot.png)

#### Prerequisite

*   [node.js](http://nodejs.org/)
*   [gulp.js](http://gulpjs.com/)

#### Installation

1.  Clone the sources from git

    ```bash
    git clone https://github.com/simnalamburt/webgl-tutorial.git
    cd webgl-tutorial
    ```

1.  Install dependencies

    ```bash
    npm install
    ```

1.  Build & Play

    ```bash
    gulp
    open build/index.html
    ```
