WebGL Tutorial
========

My WebGL practice

### Installation

1.  Clone the sources from git

    ```bash
    git init https://github.com/simnalamburt/webgl-tutorial.git
    cd webgl-tutorial
    ```

1.  Build & Play

    ```bash
    make
    open index.html
    ```

### Requirements

* make
* [node.js](http://nodejs.org/)
* [LiveScript](http://livescript.net/)
