WebGL Tutorial
========

My WebGL practice

### Installation

1.  Clone the sources from git

    ```bash
    git clone https://github.com/simnalamburt/webgl-tutorial.git
    cd webgl-tutorial
    ```

1.  Install dependencies

    ```bash
    npm install
    ```

1.  Build & Play

    ```bash
    gulp
    open index.html
    ```

### Requirements

* [node.js](http://nodejs.org/)
* [gulp.js](http://gulpjs.com/)
